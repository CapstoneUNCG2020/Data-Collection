# Data-Collection
 
## Setup:
 - run `npm install`
 - create an `.env` file
   - inside the `.env` file add `API_KEY= the_api_key` which can be found <a href="https://vickz84259.github.io/lolesports-api-docs/">here<a>
 
## Running:
 - type `node .\index.js` in the command like to run, from the projects main folder.
